# dumbdetector

A website that tracks AI models over time and detects whether they've gotten dumber.

## What is this?

AI models get updated, fine-tuned, and swapped out behind the scenes. Sometimes they get worse. Dumbdetector monitors model performance over time so you can see when a model has regressed — before you lose trust in your tools.

## Features

- Track multiple AI models across providers
- Monitor performance metrics over time
- Detect regressions and flag models that have gotten dumber
- Historical performance dashboard
